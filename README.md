# Основы статистики
## Описание
Форк от [GitHub репозитория](https://github.com/KlukvaMors/basic_stat) объёмного конспекта [лекций](https://stepik.org/course/76). 
Кое-где исправлены опечатки, кое-где добавлен конспект и код на python, кое-где сделаны косметические изменения на мой вкус.

Автор лекций: **Anatoliy Karpov** 

![Saint_Karpov](img/Saint_Karpov.png)

Конспект сопровождается кодом на python 3 с использованием Jupyter Notebook для демонстрации или решения предложенных в курсе лекций задач. 

## Запуск
Для установки нужных модулей python через менеджер `pip` выполните в темринале из директории со скаченным репозиторием следующую команду
``` bash
pip install -r requirements.txt
```
